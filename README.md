# Stock Market Analysis 
Forecasts Apple stock prices for the next 30 days using historical data from (2012-2019) with ARIMA, SARIMA, VAR, XGBoost, Random Forest, and LSTM models. The project includes data preprocessing, hyperparameter tuning, and is deployed as a Streamlit app for interactive predictions.
